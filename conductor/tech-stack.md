# Tech Stack: School Bookclub Manager

## Backend & Language
- **Environment:** Node.js
- **Language:** TypeScript
- **Rationale:** Provides a robust, type-safe environment for building scalable web applications with a massive ecosystem of libraries.

## Frontend
- **Framework:** Vue.js
- **Styling:** Vanilla CSS (as per project defaults)
- **Rationale:** Vue.js offers an approachable yet powerful framework for building reactive user interfaces, ideal for a student-focused tool.

## Database & Authentication
- **Provider:** Supabase (PostgreSQL)
- **Rationale:** Combines the reliability of PostgreSQL with an easy-to-use BaaS (Backend as a Service) layer for real-time data and authentication, speeding up development significantly.

## Infrastructure & Hosting
- **Platform:** Vercel or Netlify
- **Rationale:** Offers seamless deployment and excellent performance for modern web frameworks, with great support for Vue.js and TypeScript environments.
